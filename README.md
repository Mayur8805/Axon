# Axon

> A minimal and user-friendly terminal scraper that lets you search and download almost anything from the internet.

Axon is a Python-based TUI (Terminal User Interface) tool designed for fast media downloading directly from your terminal.

Instead of opening multiple websites, dealing with ads, popups, and fake download buttons, Axon provides a clean searchable interface where you can instantly find and download content.

---

## Features

- 🎵 Download audio from YouTube & YouTube Music
- 🎬 Download videos from YouTube
- 🖼 Download images from DuckDuckGo
- 📄 Download PDFs from PDFDrive
- ⚡ Fast keyboard-driven workflow
- 🧠 Search-based interface
- 📂 Automatic file organization
- 🖥 Minimal hacker-style TUI
- 🚀 Built for speed and simplicity

## Why Axon?

Downloading content from the internet usually means:

- searching multiple websites
- dealing with ads
- fake buttons
- slow workflows

Axon simplifies the entire process into:

```text
Search → Select → Download
```

Everything happens directly inside the terminal.

## Installation

Clone the repository:

```bash
git clone https://github.com/Mayur8805/Axon.git
cd Axon
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Axon:

```bash
python main.py
```
